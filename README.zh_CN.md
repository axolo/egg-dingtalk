# egg-dingtalk

## 安装

```bash
npm i @axolo/egg-dingtalk --save
```

## 开启插件

```js
// config/plugin.js
exports.dingtalk = {
  enable: true,
  package: '@axolo/egg-dingtalk',
};
```


## 详细配置

```js
// {app_root}/config/config.default.js
exports.dingtalk = {
};
```

请到 <https://github.com/axolo/node-dingtalk> 获取更多配置说明。

请到 [config/config.default.js](config/config.default.js) 查看详细配置项说明。
